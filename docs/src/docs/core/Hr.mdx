## Inset

You can add inset to the Hr. Inset can further be applied to different directions including `left, right, middle`.

<HrInsetDemo />

## Sizes

Hr has predefined sizes: xs, sm, md, lg, xl.
You can pass size as number and it will be set as height in px:

<HrSizesDemo />

You can get predefined sizes by importing `HR_SIZES`:

```tsx
import { HR_SIZES } from '@mantine/core';
```

<DataTable
  head={['Prop value', 'Height']}
  data={Object.keys(HR_SIZES).map((size) => [size, `${HR_SIZES[size]}px`])}
/>
